Bulk Product Upload

Quickly add products to Hadi Books Store using JSON data

Note: This tool connects directly to your backend at localhost:4000. Make sure your server is running before uploading products.

How to Use

Required Fields:

  • name (max 255 characters)
  • description (max 1000 characters)
  • price (number greater than 0)
  • category
  • author
  • language

Optional Fields:

  • originalPrice (number greater than 0)
  • isbn
  • subCategories (array or JSON string)

JSON Template:

{
  "name": "Book Name",
  "description": "Book description...",
  "price": 19.99,
  "originalPrice": 24.99,
  "category": "Fiction",
  "subCategories": ["Novel", "Adventure"],
  "author": "Author Name",
  "isbn": "1234567890",
  "language": "English"
}
